Welcome to the Model Performance Analysis

This application provides a comprehensive analysis of a machine learning model's performance in classifying data into two categories: Negative and Positive. Below are the metrics you can expect to see:

  • Confusion Matrix: A heatmap visualization that shows the number of correct and incorrect predictions categorized by actual and predicted values.
  • Classification Report: Detailed performance metrics like precision, recall, and F1-score for each category, along with overall accuracy.
  • Actual vs Predicted - Stacked Bar Chart: A stacked bar chart that displays the distribution of actual and predicted counts for each category.

Select a ticker from the dropdown below to proceed to the performance analysis.
